# Syl'slab

Syl'slab 是一套基于 Hermes Agent 的任务模式与 skill 管理系统。

## 核心模式

| 模式 | 图标 | 说明 |
|------|------|------|
| **开发者模式** | 💻 | 每步展示修改后的完整文件，确认后再继续 |
| **巡航者模式** | 👁️ | 分阶段执行，自动检测输出上限，到边界优雅暂停 |
| **长任务模式** | 🏃 | 设好蓝图自动执行，自动桥接上下文，完成后通知 |

## 快速开始

将 `skills/` 目录放入 `~/.hermes/tasks/<task>/agents/<agent>/skills/` 下，
或通过 `sylphid-skill-bridge` 自动发现。
